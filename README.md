# Heart Health Prediction App

A machine learning-based web application that predicts the risk of heart disease based on user health data and provides simple insights.

---

## Features

* Predicts heart disease risk using a trained ML model
* Accepts user inputs such as age, blood pressure, cholesterol, etc.
* Provides real-time prediction
* Built with a simple and interactive Streamlit interface

---

## Tech Stack

* Python
* Scikit-learn
* Pandas, NumPy
* Streamlit

---

## How it Works

1. User enters health parameters
2. Data is processed and passed to the trained model
3. The model predicts the risk level
4. Results are displayed instantly

---

## Run Locally

```bash
git clone https://github.com/prajwalhp15/heart-health-app
cd heart-health-app
pip install -r requirements.txt
streamlit run app.py
```

---

## Use Case

Helps in early awareness of heart disease risk using machine learning predictions.
